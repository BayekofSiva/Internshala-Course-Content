YouTube Clone - MERN Stack Project

Capstone Project: Develop a YouTube Clone Using the MERN Stack (400 marks)

Project Overview

Create a full-stack YouTube clone where users can view and interact with videos. This project will help you understand how to build a real-world application using MongoDB, Express, React, and Node.js (MERN stack).

Technologies

MongoDB Express.js React Node.js JWT Authentication Axios React Router

Next Steps - Task One

For the first task, we need to focus on setting up the foundation of the project:

  • Initialize the project repository with a clear folder structure
  • Set up the backend with Express.js and necessary middleware
  • Set up the frontend with Create React App and necessary dependencies
  • Design database schemas for Users, Videos, Channels, and Comments
  • Create a basic Express server that connects to MongoDB

This foundation will allow us to build all the required features systematically.
