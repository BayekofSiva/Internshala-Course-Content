1. Clone & Install

cd shoppyglobe
npm install

2. Environment Variables

Create a .env file in the root:

MONGO_URI=mongodb://localhost:27017/shoppyglobe

3. Start MongoDB

If installed as a service (recommended):

net start MongoDB


Or check status:

4. Seed Products
npm run seed

5. Start App
npm start


Visit http://localhost:3000.
